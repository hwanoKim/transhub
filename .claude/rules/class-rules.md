# Class Rules

## coding

| Status | 설명 | 행동 가이드 | 완료 후 |
|--------|------|-------------|---------|
| research | 사전 조사 | 구현에 필요한 기술/라이브러리/사례 조사. 조사 결과를 Notes에 기록 | → planning |
| planning | 설계/계획 | 코드 분석, 구현 방향 결정, Progress에 세부 항목 작성 | → implementation |
| implementation | 구현 | 코드 작성, 테스트. 커밋 해시 기록 | → review |
| review | 사용자 리뷰 대기 | **사용자 확인 필요**. 변경 사항 요약 보고 → 대기 | → closed (사용자 승인 시) |

research → planning → implementation → review → closed

## research

| Status | 설명 | 행동 가이드 | 완료 후 |
|--------|------|-------------|---------|
| research | 자료 수집 | 웹 검색, 공식 문서, 사례 조사 | → analysis |
| analysis | 분석/정리 | 수집 자료 비교/판단. **사용자 확인 필요** | → closed (사용자 승인 시) |

research → analysis → closed
